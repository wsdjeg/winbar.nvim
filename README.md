## Install

1. Using `winbar.nvim` in SpaceVim:

```toml
[[layers]]
  name = 'core'
  enable_winbar = true
```

2. Using `winbar.nvim` without SpaceVim:

```
Plug 'wsdjeg/winbar.nvim'
```
